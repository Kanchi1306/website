2. Car Rental System in C++


This is a C++ console-based Car Rental System that allows users to book cars and admins to view customer data. It includes features like car selection, rent calculation by hours or days, and customer data validation.

🛠 Features
Admin login with username & password
View booked car details (admin)
Car selection (Tesla, Prado, Hillux, City)
Rent calculation by hours or days
Input validation (name, phone, Aadhar, etc.)
Booking information saved to a file (Cars Info.txt)
Console-based invoice printing
